# Data Manifest

This project keeps source code under `src/` and stores raw inputs, intermediate
artifacts, and outputs in sibling directories at the repository root. These
data folders contain large proprietary files that should not be committed to
git (additions to `.gitignore` enforce this). The tables below summarize the
expected contents and how they are produced.

## `Input/`

| Subfolder  | Key files                                                                                              | Source / how to recreate                                                                                                    |
| ---------- | ------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------- |
| `DLEU/`  | `Compustat_annual.dta`, `Compustat_quarterly.dta`, `macro_vars_new.xlsx`, `macro_vars_new.dta` | Pull via `src/PyMarkup/0.0 Download Compustat.py`; update macro variables with `src/PyMarkup/macro_var_calculation.py`. |
| `CPI/`   | `CPI_annual.csv`, `CPI_quarterly.csv`                                                              | Generated by `src/PyMarkup/0.1 Download CPI.py`.                                                                          |
| `PPI/`   | `pc.data.0.Current.txt`, `PPI_annual.csv`, `PPI_quarterly.csv`                                   | `src/PyMarkup/0.2 PPI Data Preparation.py` now downloads `pc.data.0.Current.txt` and produces the CSVs.                 |
| `Other/` | `NAICS_2D_Description.xlsx`, auxiliary lookup tables                                                 | External reference spreadsheets copied locally.                                                                             |

## `Intermediate/`

Artifacts generated while running the pipeline. Examples:

- `data_main_upd_trim_1*.dta` – deflated Compustat panel created by `Create_Data.do` (or its Python port).
- `theta_W_s_window_*.dta`, `theta_c_*.dta`, `theta_acf_*.dta` – coefficient outputs from `Estimate_Coefficients` / `Estimate_Coefficients.py`.
- `main_annual.csv`, `main_quarterly.csv` – merged datasets created by the “Create Main Datasets” step.
- Figure/table staging CSVs under `Intermediate/For Figure 1/`, etc.

These files can always be regenerated by rerunning `run_all.py` (once all inputs
are in place).

## `Output/`

Contains published tables, figures, and any exports meant for papers/slides
(for example `Output/Summary Statistics/` and policy experiment plots). All of
these are derivatives of `Intermediate/` datasets and should be regenerated on
each run rather than version-controlled.
